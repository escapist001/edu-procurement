04С кем · заказчики

Выводы · пересчёт под фильтр

Методология. Запрос «учебное оборудование» (44-ФЗ + 223-ФЗ) в расширенном поиске ЕИС; парсер проходит выдачу и извлекает поля из карточек. Регион — эвристика по названию заказчика (возможны неточности). Ценовые сегменты — по порогам 44-ФЗ (котировки ≤ 3 млн, малые закупки ≤ 600 тыс.). Выборка — снимок ~1 000 свежих записей, не вся генеральная совокупность.
Стек: parser.py (requests + BeautifulSoup) · analysis.py (pandas) · дашборд — ванильный JS + Chart.js, всё считается в браузере из rows.json.
